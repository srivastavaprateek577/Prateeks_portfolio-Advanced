# Prateek Srivastava - Portfolio Website

![Thumbnail](images/Thumbnail.png)

A modern and responsive portfolio website showcasing my journey as a Full-Stack Developer. Built with Next.js and styled with Tailwind CSS, this portfolio highlights my projects, skills, and professional experience. It features a clean, dark-themed design with smooth animations powered by Framer Motion.

## Features

- **Dynamic Content**: Easily customizable via `config.js`.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Dark Theme**: A sleek and visually appealing dark mode interface.
- **Smooth Animations**: Enhanced user experience with Framer Motion animations.
- **Project Showcase**: Dedicated section to display projects with descriptions and tech stacks.
- **Skills & Experience**: Organized layout for showcasing technical skills and work history.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [Git](https://git-scm.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/srivastavaprateek577/portfolio-website.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd portfolio-website
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Configuration

This portfolio is designed to be easily customizable. You can update your personal information, projects, skills, and social links by modifying the `config.js` file located in the root directory.

Example `config.js` structure:
```javascript
export const config = {
    developer: {
        name: "Prateek Srivastava",
    },
    social: {
        github: "srivastavaprateek577",
        // ...
    },
    projects: [
        // Add your projects here
    ],
    // ...
}
```
